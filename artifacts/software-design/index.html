Exception Handling with Error Logging

This project comes from my Secure Coding course, where I worked on building better error handling using C++ exception management. The enhanced version includes a custom exception class, an isolated error logger, and clear separation of logic to make debugging and maintenance easier.

My original file was a basic demonstration, but I restructured it to better reflect real-world software practices. I added a test function, cleaner comments, and improved readability throughout. The idea wasn’t just to fix what was there—it was to level it up into something that could live in a real application with better logging and fault isolation.
